# Button

```
include { button } from components
```

```js
/**
 * @param { Object } options
 * @param { Object } attrs
**/

+component-button(options, attrs)
```

### Options

#### title

```js
{
  title: String,
  type: String,
  size: String
}
```

#### type

```
Default: 'primary'
```

#### size

```
Default: ''
```

### Attrs
Any HTML attribute
