# Button

## Подключить компонент
```
include { button } from components
```

## Вызвать компонент
```pug
+component-button(options, attributes)
```

### Options ```js Object```

#### title

```
Default: ''
```

#### type

```
Default: 'primary'
```

#### size

```
Default: ''
```

### Attributes
Any HTML attribute
